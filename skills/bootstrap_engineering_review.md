# Bootstrap: Cross-Cutting Engineering Review Skills

Version: 1.0 | Last Updated: 2026-04-11

## Purpose

Use this entry point for review quality across code, packaging/build configuration, and release-impact risk assessment.

## Apply These Skill Areas

- `skills/engineering_review/`

## Priority Order

1. `skills/engineering_review/risk_based_review.md`
2. `skills/engineering_review/code_review_checklist.md`
3. `skills/engineering_review/review_comment_response.md`

## Execution Defaults

- Present findings by severity and behavioral risk first.
- Require evidence for behavior changes and mitigation claims.
- Track unresolved risks explicitly before merge.
