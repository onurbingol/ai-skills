# Bootstrap: Cross-Cutting Contributor Workflow Skills

Version: 1.0 | Last Updated: 2026-04-14

## Purpose

Use this entry point for author-side workflow quality from branch creation through PR readiness.

## Apply These Skill Areas

- `skills/contributor_workflow/`

## Priority Order

1. `skills/contributor_workflow/branching_and_change_scope.md`
2. `skills/contributor_workflow/commit_hygiene_and_messages.md`
3. `skills/contributor_workflow/pull_request_hygiene_and_review_readiness.md`

## Execution Defaults

- Keep branches focused on one coherent change intent.
- Prefer incremental commits that are easy to review and revert.
- Treat PR readiness as a validation state, not a draft narrative.

## Boundary Notes

- This pack is author-side only.
- Reviewer decision frameworks remain under `skills/engineering_review/`.
