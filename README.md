# AI Agent Skills

## Introduction

This repository contains curated skill definitions, bootstraps, and workflow guidance
for AI-assisted software engineering across development, testing, documentation,
debugging, engineering review, and delivery engineering.

## How to Use

- Start from [AGENTS.md](AGENTS.md) to understand routing and configuration.
- Use [skills/bootstrap.md](skills/bootstrap.md) to find the canonical bootstrap entry points.
- Use [SKILLS.md](SKILLS.md) if you want to create or maintain skills.

## Scripts

- See [scripts/README.md](scripts/README.md) for brief usage of repository scripts.
